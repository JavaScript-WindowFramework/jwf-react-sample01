# @jswf/react

React用仮想Windowコンポーネント

## 内容

Reactで仮想ウインドウを実現するためのコンポーネント

### 現時点で使用可能な機能

- ウインドウの移動
- リサイズ
- 最大化
- 最小化
- 重ね合わせ
- 親子ウインドウ
